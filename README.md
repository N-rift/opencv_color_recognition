# OpenCV Color Recognition

## Overview

This project uses OpenCV to detect and recognize colors through a webcam feed.

The program identifies colored objects and displays their color name on the screen.

## Features

- Real-time color detection
- Bounding box around detected objects
- Color label display
- OpenCV-based image processing

## Requirements

- Python 3
- OpenCV
- NumPy

- color_detection.py
